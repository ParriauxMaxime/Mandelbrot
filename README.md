# Mandelbrot

Just some play time with svg html element.

## Installation

```bash
git clone https://github.com/ParriauxMaxime/Mandelbrot.git
yarn #or npm i
```

## Usage

```bash
yarn start
#go to localhost:3000
```

OR

```bash
yarn build && yarn serve
#go to localhost:5000
```
